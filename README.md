# LibreNMS Network Automation Scripts

ဒီ Project ဟာ LibreNMS Monitoring System နဲ့ ချိတ်ဆက်ပြီး Network Infrastructure တွေကို Automation လုပ်ဖို့နဲ့ Monitoring အချက်အလက်တွေကို စုစည်းဖို့အတွက် ရည်ရွယ်ပါတယ်။

## Features
* LibreNMS API ကို အသုံးပြု၍ Device data များ ရယူခြင်း။
* Network devices များ၏ Configuration backup များကို အလိုအလျောက် ရယူခြင်း။
* Monitoring alerts များကို စစ်ဆေးခြင်း။
